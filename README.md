# hashtag

Goal app

    Propose an app for people who want to associate picture to  hastag on social network
    
How is it work ?

    -  Go on the site  
    -  Upload  your picture
    -  Select best hastag for your post
    
Start app

    docker-compose up

Front is running on port 5000

## Directories

- front  
    Front of the app
- selector  
    Api for hashtag recommendation
- detect  
    image recognition
